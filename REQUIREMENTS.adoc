== 1. Introducción

=== 1.1 Propósito

Este documento describe los requerimientos funcionales y no funcionales del sistema *Splitify*, una aplicación web que permite a los usuarios categorizar y organizar sus canciones de Spotify por idioma, género musical y época/década de lanzamiento.

=== 1.2 Alcance

Splitify es una aplicación web construida con Spring Boot que se integra con la API de Spotify (OAuth) y la API de ChatGPT. Permite a los usuarios seleccionar playlists existentes (incluyendo Liked Songs) y generar nuevas playlists organizadas según criterios de idioma, género y/o época.

=== 1.3 De finiciones y Acrónimos

|===
|Término |Definición 

|Liked Songs |Playlist especial de Spotify con las canciones marcadas como favoritas por el usuario 
|ChatGPT API |API de OpenAI utilizada para clasificar el género musical de cada canción 
|SRS |Software Requirements Specification 
|OAuth |Protocolo de autorización utilizado para la autenticación con Spotify 
|===

'''

== 2. Descripción General

=== 2.1 Perspectiva del Producto

Splitify actúa como intermediario entre el usuario y Spotify. No almacena música; únicamente lee las playlists del usuario, clasifica las canciones y crea nuevas playlists en la cuenta de Spotify del usuario.

=== 2.2 Usuarios Objetivo

Usuarios de Spotify que desean organizar automáticamente sus bibliotecas musicales según idioma, género o década sin hacerlo manualmente.

=== 2.3 Restricciones

* Requiere una cuenta activa de Spotify.
* La clasificación de género depende de la disponibilidad de la API de ChatGPT.
* El despliegue objetivo es en Linode.
'''

== 5. Requerimientos Funcionales

=== RF1 — Autenticación con Spotify

*Descripción:* El sistema permitirá al usuario iniciar sesión mediante Spotify OAuth y cerrar sesión.

*Criterios de aceptación:*
- El usuario puede autenticarse con sus credenciales de Spotify.
- El usuario puede cerrar sesión en cualquier momento.
- La sesión se mantiene activa mientras el usuario navega la aplicación.

'''

=== RF2 — Visualización de Playlists

*Descripción:* El sistema obtendrá y mostrará todas las playlists del usuario (incluyendo Liked Songs) en la página principal, en la parte superior mostrara las playlists ya creadas por el usuario en spotify, y la parte inferior, las creadas con Splitify. El usuario puede entrar en cada playlist(ya existente creada en Spotify por el usuario o creada con Splitify) para ver las canciones con su información. Las playlists creadas con Splitify tienen la opcion de poder ser eliminadas, y actualizarse al ultimo contenido(en el caso de que las playlists que se usaron para crear esa nueva playlist, se le hayan agregado mas opciones)

*Criterios de aceptación:*
- Se muestran todas las playlists del usuario en la página principal.
- La playlist de Liked Songs se incluye en la lista.
- Al seleccionar una playlist, se muestran las canciones con su información (nombre, artista, año, género).
- Las playlist creadas con Splitify tienen la opcion de "Eliminar" y "Actualizar".

'''

=== RF3 — Obtención de Metadatos de Canciones

*Descripción:* El sistema obtendrá la información de cada canción en una playlist: id, nombre, artista, año original de lanzamiento y género (clasificado mediante ChatGPT API).

*Criterios de aceptación:*
- Se obtienen todos los campos definidos en la sección 4.1 para cada canción.
- El género se clasifica correctamente en una de las 10 categorías definidas en la sección 3.2.

'''

=== RF4 — Ordenar por Idioma

*Descripción:* El usuario podrá crear nuevas playlists filtrando canciones por idioma.

*Flujo:*
1. El usuario selecciona una o más playlists origen (incluyendo Liked Songs).
2. El sistema crea las nuevas playlists en Spotify, creara tantas playlists como idiomas soportados existan en tus musicas(son 15 y 16 incluyendo "Sin letra").
3. En el caso de que haya musicas de idiomas no soportados, a menos que no excedan las 5 canciones(donde se les creara igualmente una nueva playlist individual), se pondran en una playlist llamada "Other languages"
4. Las musicas estaran ordenadas en las playlists por fecha original de lanzamiento

*Nomenclatura de playlists creadas:*
`Splitify {Idioma} Songs` (ej: `Splitify Español Songs`, `Splitify Inglés Songs`)

'''

=== RF5 — Ordenar por Género

*Descripción:* El usuario podrá crear nuevas playlists filtrando canciones por género musical.

*Flujo:*
1. El usuario selecciona una o más playlists origen (incluyendo Liked Songs).
2. El sistema crea las nuevas playlists en Spotify, creara tantas playlists como generos soportados existan en tus musicas(son 10).
3. Las musicas estaran ordenadas en las playlists por fecha original de lanzamiento

*Nomenclatura de playlists creadas:*
`Splitify {Género} Songs` (ej: `Splitify Rock Songs`)

'''

=== RF6 — Ordenar por Época/Década

*Descripción:* El usuario podrá crear una nueva playlist incluyendo absolutamente todas las musicas de las playlist que selecciono, y ordenarlas en una sola por orden de lanzamiento original.

*Flujo:*
1. El usuario selecciona una o más playlists origen (incluyendo Liked Songs).
2. El sistema crea la nueva playlist en Spotify.

*Nomenclatura de playlists creadas:*
`Splitify Songs By Release Date (Playlists included: {Playlists})` (ej: `Splitify Songs By Release Date (Playlists included: Liked Songs, Rock Classics})`)

'''

=== RF7 — Ordenar por Idioma + Género + Época (Personalizado)

*Descripción:* El usuario podrá crear una nueva playlist combinando filtros de idioma, género y época simultáneamente.

*Flujo:*
1. El usuario selecciona una o más playlists origen (incluyendo Liked Songs).
2. Selecciona idioma(s), género(s) y época(s).
3. El usuario escribe el nombre que desea para la playlist.
4. El sistema crea una única playlist con las canciones que cumplan todos los criterios.

*Nomenclatura de playlist creada:*
(El usuario pone un nombre)

'''

== 6. Reglas de Negocio

|===
|# |Regla 

|RN1 |Todas las playlists creadas por el sistema estarán ordenadas por fecha de lanzamiento de las canciones (ascendente). 
|RN2 |Las canciones cuyo idioma no coincida con ninguno de los 16 soportados se agrupan en "Other Languages" cuando se filtra por idioma (a menos de que existan mas de 5 canciones con ese idioma no soportado, cuyo caso igualmente se le creara una nueva playlist individual).
|RN3 |La clasificación de género se delega a ChatGPT API, mapeando cualquier subgénero a las 10 categorías principales. 
|===
